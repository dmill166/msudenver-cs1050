# Assignments

Programming assignment prompts, specifications, and starter code — the larger graded deliverables.

## Convention

- One folder per assignment: `aNN_topic/`
- Each assignment folder contains:
  - `README.md` — problem statement, requirements, grading criteria (point allocation), submission format, and the ABET/CS2023 outcome(s) assessed
  - `starter/` — starter code / templates (MIT)
  - `tests/` — public test cases students can run (private/hidden tests live in the solutions repo)
- Solution keys, rubrics, and autograders live in the **private** `msudenver-cs1050-solutions` repo.

## Assessment mix (per course-of-record)

CS 1050 assesses via programming assignments, quizzes/exams, a final, **plus research papers/book reports and oral presentations**, with a written-communication emphasis. Reserve a track here for the writing/presentation deliverables, not just code.
