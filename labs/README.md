# Labs

Guided, in-class lab prompts and starter code. Labs are short, scaffolded exercises that reinforce the day's lecture; assignments (in [`../assignments/`](../assignments/)) are the larger graded deliverables.

## Convention

- One folder per lab: `labNN_topic/`
- Each lab folder contains:
  - `README.md` — the prompt, learning goals, and the ABET/CS2023 outcome(s) it exercises
  - `starter/` — starter code students begin from (PolyForm Noncommercial)
  - (Solutions live in the **private** `computer-science-1-solutions` repo, never here.)

## Writing/presentation thread

CS 1050's course-of-record emphasizes written communication and oral presentation. Tag labs that carry that thread (e.g., a code-walkthrough write-up or a short demo) so the skill is built deliberately, not bolted on.
